Scope — exact deliverables
Backend
File: new server/lib/oauth/connections.ts

Two route handlers, mounted in server/lib/oauth/index.ts:

GET /api/oauth/connections — returns { connections: Connection[] } for the authenticated user's active membership. Connection = { clientId: string, clientName: string, scopes: string[], firstAuthorizedAt: ISO8601, lastUsedAt: ISO8601 | null, activeTokenCount: number }. Query: SELECT from oauthClients JOIN oauthTokens ON client_id, WHERE oauth_tokens.membership_id = :membershipId AND oauth_tokens.revoked_at IS NULL, GROUP BY client_id. firstAuthorizedAt = MIN(issued_at), lastUsedAt = MAX(issued_at), activeTokenCount = COUNT(*). scopes = distinct values from the matching oauth_tokens.scope strings, split on space.
DELETE /api/oauth/connections/:clientId — revokes all non-revoked tokens for (clientId, membershipId) in a single UPDATE. Returns { revokedCount: number }. Logs to mcp_audit_log with tool_name = "oauth.revoke", client_id = :clientId, no audience. Idempotent: revoking when no active tokens returns { revokedCount: 0 } and 200, not 404.
Mount both in server/lib/oauth/index.ts after the existing routes.
