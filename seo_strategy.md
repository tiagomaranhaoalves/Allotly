# SEO Strategy

## In scope
- Public marketing pages
- Documentation pages
- Legal/compliance pages
- Public contact and company pages

## Out of scope
- Authenticated dashboard (`/dashboard/**`)
- Admin pages (`/admin/**`)
- OAuth consent and account flows used after authentication or app authorization
- API endpoints except where they directly affect crawlability (`robots.txt`, `sitemap.xml`, `llms.txt`, static file routing)

## Target audience
- Teams and operators managing AI API access, budgets, and model permissions
- Workshop, education, and voucher-based AI access distributors

## Primary keywords
- AI API budget controls
- AI spend management
- API key budget enforcement
- AI access management
- AI vouchers
